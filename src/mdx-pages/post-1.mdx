# デザインとフォント

これを見りゃわかるように、ダークテーマ、高彩度、グラデーション、透明、すりガラスがすき。ダークテーマのあるアプリなら有無を言わさずダークにする。
__服でもデバイスでも色選択させると9割黒になる__。

フォントはモダン志向。町中で新ゴ発見芸人だが新ゴは持っていない。
ほかのフォントもひと目で区別がつくようにしたい。

ちなみにこのサイトの背景は「[Hitchhiker](https://www.1001fonts.com/hitchhiker-font.html)」というフリーフォント。

# ブラウザ

最初がChromeで、次がVivaldiだったはず。SleipnirやOperaとかも試したけど、やFN1だった。___IEは論外___。

# プログラミング

一番使ってるのはJavaScript/TypeScript。ただまじめにチュートリアルをやっているか怪しい雰囲気で書いており、危険。React大好き。

他にやりたい言語があるとすれば、KotlinとRustあたりだろうか。Kotlinはお察しの通りAndroidアプリ、Rustはなんとなく。
