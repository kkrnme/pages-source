# 手段一覧

|手段|オヌヌメ度|拠点の多さ|手間|
|-|-|-|-|
|クレジット/デビット|高い|どこでも|かからない|
|セブン銀行ATM|次点|かなり多い|あんまり|
|銀行ATM(Pay-easy)|ダメ|少ない~そこそこ|クソ|

以上の通り__クレジットかデビットを登録するのが一番早いです__。
これより下は各手段の解説になります。結論だけ必要な方はここでお帰りください。

# クレジット/デビット

Kyash自身が一番推してるチャージ手段です。自動チャージを有効にすればチャージの必要はなくなりますし、自動チャージでなくとも家でもどこでもチャージできるのは利点の塊です。

当然ですがクレジットもデビットもない場合は使えません。
[Kyashの利用規約](https://kyash.co/tos/)を見る限り年齢には下限がないようですが、デビットの作れない15歳未満ではたぶんこの手段は不可能だと思います。
逆に言えばKyashユーザーは15歳になった瞬間に銀行の口座をブチ開け、デビットを発行して連携しなければならない(MUST)とも言えます。

なおVISAがほしいからといって__間違ってもゆうちょ銀行に行ってはいけません__。ゆうちょのデビット/プリペイドであるmijicaはプリペイドが付いているばかりに__Kyashから拒否られます__。
サポートに問い合わせた上でNGを食らっているので確実に無理です。

# セブン銀行ATM

クレジット/デビットが使えないときのための次点です。

このご時世セブンイレブンなんてそこら中に転がってる(首都圏民の偏見です)のでチャージしたいと思えばだいたいいつでもチャージできます。
さらになぜか関係ないスーパーに置いてあったりします。もしかしたらイトーヨーカドーにも置いてあるのかも？

デビットの使えない15歳未満のKyashユーザーにとっては主要なチャージ手段になると思われます。

# 銀行ATM(Pay-easy)

銀行のATMからチャージできますが、__入金が3000円単位__ なのがかなり痛いです。
