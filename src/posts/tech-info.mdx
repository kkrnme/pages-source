# MDX とは？

**React コンポーネントを使える Markdown。JS に対する JSX。TS に対する TSX。**

試しにこのスライダーを動かしてみてほしい。右に動かせば広く、左に動かせば狭く、文字と文字の間隔が広がったり狭まったりする。
このスライダーは横の数字と併せて**React コンポーネント**であり、**Emotion がスタイルを編集している**。
